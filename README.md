🍽️ Apna Dhaaba – Recipe Project

This is a simple and visually appealing Recipe Web Page built using HTML and CSS.
The project displays popular Indian sweets with their ingredients and step-by-step recipes.

📌 Project Overview

“Aapna Dhaaba” is a beginner-friendly project that focuses on:

-Structuring a webpage using HTML
-Styling using CSS
-Creating a clean and responsive layout
-Displaying recipes in an organized format

🚀 Features

-🧭 Navigation bar (Home, Contact, About)
-🍬 Two recipe sections:
--Kaju Katli
--Motichoor Ladoo
-📋 Ingredients list for each recipe
-📝 Step-by-step cooking instructions
-🖼️ Image hover effect (zoom + overlay)
-🎨 Gradient backgrounds and modern UI

🛠️ Technologies Used
-HTML5 – For structure
-CSS3 – For styling and layout
